SAE-309 | Projet individuel

Projet PWA — Application web progressive

Contexte BUT Réseaux & Télécommunications

Développement d’une application web progressive (PWA) simulant un environnement de travail informatique. L’application intègre un éditeur Python, plusieurs terminaux simulés (Python, Linux, Windows) ainsi que l’utilisation d’API externes, dont une intelligence artificielle intégrée pour assister l’utilisateur dans l’écriture et la compréhension du code. Les données sont conservées via le localStorage et un service worker permet le fonctionnement hors ligne. Ce projet m’a permis d’approfondir la conception d’une PWA, la gestion du cache, le stockage local et l’intégration d’API externes.

HTML / CSS JavaScript PWA Service Worker localStorage
